This project is a real-time chat application built using Node.js, Express, and Socket.IO.
It demonstrates how WebSockets enable two-way communication between the client and server, allowing instant message delivery without page refresh.

Socket.IO is built on top of WebSockets and provides additional features such as:

Automatic reconnection

Fallback transports

Event-based communication

Room and namespace support

This project is a simple implementation to help understand real-time communication in web applications.
